分享一下个人使用过感觉比较好的IDEA插件
包括提高效率，以及美化编辑器
运行环境： IntelliJ IDEA 2024.3.3 (Ultimate Edition)

插件列表

插件名称 描述 插件主页
.ignore VCS忽略名单 JetBrains
Alibaba Java Coding Guidelines 阿里巴巴编码规范 XenoAmess

CamelCase 字母大小写切换 NetNexus
Chinese (Simplified) Language Pack 中文语言包（已内置） JetBrains
CheckStyle-IDEA 代码检查 Jamie Shiell
CodeGlance Pro 代码地图 Nasller
Gitee 添加对Gitee的支持 码云
Grep Console 控制台日志调色，针对不同的日志级别，使用不同的颜色，方便查看日志 Vojtech Krasa
JProfiler 配合JProfiler软件进行调试用 ejtech
Json Assistant Json相关工具集合 Memory
MyBatis Log EasyPlus 把控制台的SQL日志转换成完整的SQL语句 linkkou
MyBatisPlus 生成CRUD代码、以及MyBatis相关操作 MybatisPlus
RestfulBox - API 一套 RESTful 服务开发辅助工具集 RestfulBox
